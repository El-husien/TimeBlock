# TimeBlock - Focus & Productivity Dashboard

A beautiful focus and productivity tool with time tracking, Pomodoro timer, and per-site limits.

## 🌐 Web Demo (GitHub Pages)

**Live Demo:** https://el-husien.github.io/TimeBlock/

This is a showcase/demo version of the dashboard - fully functional UI with sample data.

## 🧩 Chrome Extension (Full Version)

For full functionality with actual time tracking, install the Chrome extension:

1. Download the source code
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this folder

## Features

### ⏱️ Pomodoro Timer
- Work/break cycles with customizable durations
- Visual timer with phase indicators
- Auto-start options

### 🎯 Focus Mode
- Quick focus sessions (15m, 25m, 60m)
- Block distracting websites
- Timer with remaining time

### 📊 Analytics
- Time tracking by category
- Weekly trends
- Top sites overview
- Export data as CSV

### 🚫 Site Limits
- Set daily limits per site
- Visual progress bars
- Auto-block when exceeded

## Files

- `index.html` - Web demo (works on GitHub Pages)
- `popup.html/js` - Chrome extension popup
- `dashboard.html/js` - Full dashboard
- `background.js` - Extension engine
- `styles.css` - All styling
